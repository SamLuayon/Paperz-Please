# Paperz-Please (Winx Edition)

A simple, creative Java Swing / JOptionPane game developed as a final project for my **Introduction to Computing** course during my first year. 

It is heavily inspired by the indie game *Papers, Please*, but reimagined with a magical *Winx Club* theme!

##  About the Project
Since this was built during my first year, the mechanics rely on Java's built-in `JOptionPane` dialogs and basic standard logic. To elevate the visual experience and match the theme, the game utilizes custom background images and graphics for character interaction and data verification.

## How to Run the Game
To play the game correctly from the source files, follow these steps inside your IDE (like Apache NetBeans):

1. Clone or download this repository.
2. Open the project in your Java IDE.
3. Navigate to the source files and open **`MainMenu.java`**.
4. Right-click inside `MainMenu.java` and select **Run File** (or press `Shift + F6`).

##  Tech Stack & Features
* **Language:** Java
* **GUI Component:** `javax.swing.JOptionPane` (enhanced with custom `ImageIcon` graphics)
* **Theme:** Winx Club inspired assets and documentation checking
